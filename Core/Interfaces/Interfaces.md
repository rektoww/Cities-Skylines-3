# Документация по интерфейсам (Interfaces)

Этот документ описывает интерфейсы, используемые в симуляции города.

## Глава 1: IConstructable.cs

Интерфейс `IConstructable` определяет контракт для всех объектов, которые могут быть построены в игре. Он использует **статические абстрактные члены** для получения метаданных о строительстве (стоимость, материалы), что позволяет получать эту информацию напрямую из класса, не создавая его экземпляр.

### Статические абстрактные свойства

Эти свойства должны быть реализованы как `static` в каждом классе, реализующем интерфейс.

- **`BuildCost`**: `static abstract decimal`
  - Получает стоимость постройки здания.

- **`RequiredMaterials`**: `static abstract Dictionary<ConstructionMaterial, int>`
  - Получает словарь, содержащий перечень необходимых строительных материалов и их требуемое количество.

### Методы экземпляра

- **`TryPlace(int x, int y, GameMap map)`**: `bool`
  - Пытается разместить экземпляр здания на указанных координатах карты. Возвращает `true` в случае успеха.
