# Документация по моделям управления персоналом (HR Models)

Этот документ описывает классы, связанные с системой управления персоналом в организации.

## Глава 1: Employee.cs

`Employee` — это класс, представляющий сотрудника в системе управления персоналом. Содержит информацию о гражданине, грейде, отделе и зарплате.

### Свойства

- **`Citizen`**: `Citizen` - Гражданин, являющийся сотрудником.
- **`Grade`**: `EmployeeGrade` - Грейд сотрудника в организации (Junior, Specialist, Senior).
- **`Department`**: `Department` - Отдел, в котором работает сотрудник.
- **`Salary`**: `decimal` - Зарплата сотрудника.

### Методы

- **`ToString()`**: `string` - Возвращает строковое представление сотрудника в формате "ID - Профессия (Грейд) - Зарплата".

## Глава 2: Department.cs

`Department` — это класс, представляющий отдел в организации. Содержит информацию о сотрудниках, бюджете и местоположении.

### Свойства

- **`Name`**: `string` - Название отдела.
- **`Type`**: `DepartmentType` - Тип отдела (Administration, Commercial, Production).
- **`Location`**: `Building` - Здание, в котором расположен отдел.
- **`Employees`**: `List<Employee>` - Список сотрудников отдела.
- **`Budget`**: `decimal` - Бюджет отдела для выплаты зарплат (по умолчанию 100,000).

### Методы

- **`CanHire(JobType profession)`**: `bool` - Проверяет возможность найма сотрудника указанной профессии (ограничение по количеству сотрудников).
- **`CanAffordSalary(decimal salary)`**: `bool` - Проверяет возможность выплаты указанной зарплаты в рамках бюджета отдела.
